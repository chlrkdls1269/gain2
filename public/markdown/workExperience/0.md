#### 삼성전자 DX 디지털 트윈 (수원 디지털 시티 보안 구역 상주)

데이터 정합성·보안이 중요한 백엔드 영역을 담당하고 있습니다.

- **DX MetaFactory (DMF)** — 삼성전자 전 법인 디지털 트윈 공장 데이터 통합 관제 플랫폼 F/E·B/E 개발 및 운영 *(상세 → Project 섹션)*
- **DT Suite Designer (DTS)** — 디지털 트윈 공장 설비 레이아웃 편집 SW 백엔드. 삼성전자 시니어 백엔드 개발자와 **2인 공동 개발**, 초기 프로젝트 구조 및 빌드 파이프라인 구성 담당
  - 동시 편집 충돌 방지 스키마 및 편집 이력 스냅샷 테이블 구축
  - **법인 / 그룹 / 개인 단위 RBAC** 권한 체계 설계
  - 편집 이벤트 기반 알림 메일링 파이프라인 구축
  - 스냅샷 기반 과거 레이아웃 / 오브젝트 조회 API 개발
