#### 역할 — 백엔드 개발 / 운영

- 삼성전자 DX 전 법인 공장의 디지털 트윈 데이터를 실시간 통합 관제하는 **엔터프라이즈 플랫폼**
- 약 **1,600명** 임직원이 사용 중이며, **이재용 회장 · 베트남 대통령 국빈 방문 시연**이 진행된 미션 크리티컬 시스템

#### 주요 기여

- **Kafka 사용량 월별 집계 알고리즘 및 대시보드** 설계·개발
  - Health Check, UTC 시간 공통화, DB Pivot 기반 집계 쿼리, 월별 요약 UI
  - 전 법인 비용 청구 근거 데이터로 활용되며 **임원진(그룹장·부사장) 정기 보고**에 사용
- **Samsung Knox SSO 연동 임직원 2차 인증 시스템** 설계 및 구현
- 디지털 트윈 공장 설비 기준정보 / 알람 정보 조회·관리 API 개발 및 유지보수
- **누적 400건 이상**의 장애·개선 VOC 처리 — 반복 장애 패턴을 분석해 운영 가이드·점검 절차를 구조화하여 유사 VOC 재발률 감소
- **Edge 브라우저 버그 단독 규명**: 내부 코드 문제로 의심되던 엑셀 다운로드 장애를 Chromium 버전별 회귀 테스트로 외부 이슈로 판명, **Microsoft 공식 수정**까지 이끌어냄 *(삼성전자 내부 VOC 시스템 공식 기록)*
- 삼성 SDS 모의 해킹 대응 보안 강화 및 TLS 인증서 운영
