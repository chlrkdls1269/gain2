#### 역할 — 모바일 앱 화면 기획 / 이미지 편집 알고리즘 개발

- SK텔레콤 **FLY AI 최종 팀 프로젝트**
- 단체사진에서 **각자 마음에 드는 얼굴**을 골라 자연스럽게 합성해주는 모바일 앱
- SK텔레콤 임원 시연 완료

#### 주요 기여

- 모바일 앱 **이미지 편집 화면 기획 및 구현**
- 선택한 사진과 모바일 갤러리 사진 사이의 **이미지 유사도 측정 알고리즘** 개발 — 유사 사진 리스트 자동 추출

- 마스크 착용 상태에서도 눈 크기 측정이 가능하도록 관련 논문을 참조해 알고리즘 개선
